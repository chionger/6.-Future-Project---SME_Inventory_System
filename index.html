SME Inventory System

Inspiration : While This is a page for me to tinkle with a base structure which could be expanded into a more sophisticated system for SMEs

Motivation : The idea is to create a database that allows the user to:

  1. keep track of date that inventory was added
  2. keep track of new inventory storage location and information
  3. future is to recognise stock via photo of item on display rack
  4. show quantity and size of remaining stock
  5. show location of remaining stock
  6. quantity and size will be be reduced when sales are made
